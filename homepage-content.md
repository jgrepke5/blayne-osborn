## GODADDY BUILDER TIPS

1. **Navigation:** Keep it simple - Home, About, Issues, Endorsements, Donate (make Donate a button-style link in contrasting color)

2. **Mobile:** Test all sections on mobile; GoDaddy should handle responsive automatically but verify text readability over images

3. **Buttons:** Use consistent styling throughout:
   - Primary (Donate): Red background, white text
   - Secondary: Navy outline or navy background, white text

4. **Images:** Upload high-resolution versions; GoDaddy will optimize

5. **Forms:** GoDaddy has built-in form builders; ensure you can add the required disclaimer text

6. **Speed:** Optimize images before upload if site loads slowly
